## 配置文件便携化

> 在exe所在目录新建一个.config目录，然后再设置一个.config/PORTABLE的空文件（需重启软件），这时配置目录将变更为exe所在目录下的.config目录。
> 这时还可以直接将~/.config/clash-verge之前的配置，直接copy到exe所在目录/.config/clash-verge。
